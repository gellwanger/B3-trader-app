# B3 Trader App


### 🔧 Instalação
```bash
# Clone este repositório
$ git clone git@github.com:gellwanger/B3-trader-app.git

# Instale as dependências
$ npm install

# Execute a aplicação
$ npm start

## 🚀 Utilizando

## ⚙️ Executando os testes
$ npm run test

```
